# My browser extension

1. Install dependencies:
```sh
npm install
```
2. Ensure `dist/` file is generated:
```sh
npm run build
```
3. Start the extension:
```sh
npm run dev:firefox # or dev:chrome
```

To debug popup:
- `about:debugging#/runtime/this-firefox` -> Inspect
